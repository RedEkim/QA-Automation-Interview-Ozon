[2] ЭТАП 1: BACKEND, СЕТИ & БАЗЫ ДАННЫХ (21.07.2026 — 25.10.2026)
--------------------------------------------------------------------------------

Недели 1-3 (21.07 - 10.08) | Сети, HTTP/HTTPS & REST:
  [ ] Изучить структуру HTTP-запроса и ответа (Start line, Headers, Body).
  [ ] Выучить ключевые заголовки: Host, Authorization, Content-Type, Cache-Control, Cookie.
  [ ] Выучить ответы и категории статус-кодов (2xx, 3xx, 4xx, 5xx).
  [ ] Разработать полное понимание методов REST (GET, POST, PUT, PATCH, DELETE).
  [ ] Разобрать понятие идемпотентности методов HTTP.
  [ ] Изучить механизмы авторизации: Cookie/Session vs JWT (Header, Payload, Signature).
  [ ] ПРАКТИКА: Погонять запросы через Postman / Insomnia по публичным REST API.

Недели 4-6 (11.08 - 31.08) | gRPC & Сетевая архитектура:
  [ ] Изучить архитектуру gRPC и Protocol Buffers (.proto-файлы).
  [ ] Понять отличия gRPC от REST (Protobuf vs JSON, HTTP/2 vs HTTP/1.1, бинарность, стриминг).
  [ ] Пошагово разложить ответ на вопрос: "Что происходит, когда вы вводите google.com в адресную строку?" (DNS -> TCP 3-way handshake -> TLS -> HTTP -> DOM/CSSOM -> Render).
  [ ] ПРАКТИКА: Пройти тренажер secret_trainer (GitHub), поработать с gRPC через BloomRPC/Postman.

Недели 7-11 (01.09 - 05.10) | Реляционные БД & SQL — PostgreSQL:
  [ ] Освоить синтаксис: SELECT, WHERE, GROUP BY, HAVING, ORDER BY, LIMIT / OFFSET.
  [ ] Закрепить все типы соединений: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, CROSS JOIN.
  [ ] Изучить подзапросы, агрегатные функции (COUNT, SUM, AVG, MIN, MAX) и CASE.
  [ ] Разработать понимание индексов (B-Tree, Hash): почему ускоряют выборку, но замедляют запись.
  [ ] Изучить транзакции и принципы ACID.
  [ ] Выучить 4 уровня изоляции транзакций и феномены чтения (Dirty Read, Non-repeatable Read, Phantom Read).
  [ ] ПРАКТИКА: Пройти SQL Academy и решить 50+ задач на sql-ex.ru.

Недели 12-14 (06.10 - 25.10) | NoSQL & Кэширование — Redis:
  [ ] Понять назначение In-Memory БД и концепцию кэширования.
  [ ] Изучить базовые структуры данных Redis (String, Hash, List, Set, Sorted Set) и параметр TTL.
  [ ] ПРАКТИКА: Развернуть Redis локально и отработать основные команды в redis-cli.

--------------------------------------------------------------------------------
[3] ЭТАП 2: ЯЗЫК ПРОГРАММИРОВАНИЯ & АВТОМАТИЗАЦИЯ (26.10.2026 — 31.01.2027)
--------------------------------------------------------------------------------

Недели 15-19 (26.10 - 29.11) | Базовый ЯП & ООП:
  [ ] Освоить конструкции языка, работу с коллекциями и структурами данных (массивы, списки, хэш-таблицы/словари, множества).
  [ ] Оценить временную сложность базовых операций над коллекциями (O(1) vs O(n)).
  [ ] Закрепить ООП: Инкапсуляция, Наследование, Полиморфизм, Абстракция (в контексте тестов).
  [ ] Освоить обработку исключений и ошибок (try-catch / try-except).
  [ ] ПРАКТИКА & ЧТЕНИЕ: Прочитать главы 1-6 книги "Чистый код" Р. Мартина, решать задачи на LeetCode (Easy).

Недели 20-24 (30.11 - 03.01) | Фреймворк автотестирования & Архитектура:
  [ ] Разобраться в жизненном цикле тестов в выбранном фреймворке.
  [ ] Освоить фикстуры / аннотации и их скоупы (test/method, class, module, session).
  [ ] Освоить параметризацию тестов, параллельный запуск и тегирование.
  [ ] Изучить паттерны: Page Object Model (POM), Page Factory, Builder.
  [ ] ПРАКТИКА: Написать с нуля фреймворк для тестирования REST API.

Недели 25-28 (04.01 - 31.01) | Интеграции & Отчетность:
  [ ] Настроить интеграцию автотестов с Allure Report / Allure TestOps.
  [ ] Реализовать логирование тестов и снятие скриншотов/дампов при падении.
  [ ] ПРАКТИКА: Покрыть тестовый REST/gRPC сервис написанными автотестами.

--------------------------------------------------------------------------------
[4] ЭТАП 3: WEB, MOBILE, GIT & DOCKER (01.02.2027 — 04.04.2027)
--------------------------------------------------------------------------------

Недели 29-31 (01.02 - 21.02) | Frontend, DevTools & Локаторы:
  [ ] Изучить DOM-дерево, построение устойчивых CSS-селекторов и XPath.
  [ ] Освоить работу с Chrome DevTools: Network (Throttling, Preserve Log), Application (Cookies, Storage), Console.
  [ ] ПРАКТИКА: Автоматизировать сценарии на the-internet.herokuapp.com.

Недели 32-34 (22.02 - 14.03) | Mobile Testing:
  [ ] Изучить особенности Android vs iOS (Activity Lifecycle vs App Lifecycle).
  [ ] Освоить работу с логами: adb logcat (Android), Console / xcrun simctl (iOS).
  [ ] Освоить сниффинг и подмену трафика через Charles / Proxyman / Mitmproxy (установка SSL-сертификатов).

Недели 35-37 (15.03 - 04.04) | Git, Linux CLI & Docker:
  [ ] Git CLI: checkout, branch, rebase, merge, cherry-pick, reset, разрешение конфликтов.
  [ ] Linux CLI: Навигация и поиск (cd, ls -la, grep, find, cat, tail -f, ps aux, kill).
  [ ] Docker: Написание Dockerfile (FROM, RUN, COPY, CMD), работа с docker-compose.
  [ ] ПРАКТИКА: Заковать свой фреймворк автотестов в Docker-контейнер и запустить через docker-compose up.
